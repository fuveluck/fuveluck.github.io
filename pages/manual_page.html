Manual: Using the First-order Logic to Prolog Converter

Overview

This converter allows you to input logical expressions in first-order logic and convert them into Prolog syntax.

It also supports LaTeX rendering for easy input of formulas.

Getting Started

Follow these steps to use the converter:

  • Navigate to the input area, where you can type or insert logic symbols using the available buttons.
  • Click the "Convert" button to transform your expression into Prolog syntax.
  • View the Prolog output in the result area below the input field.
  • If needed, click "Clear" to reset the input field.

Input Field

The input field supports LaTeX-like syntax for logical operations. You can also use the provided buttons to insert common logical symbols:

  • \Rightarrow: Implication (rendered as $$\Rightarrow$$)
  • \land: Conjunction (rendered as `\land`)
  • \lor: Disjunction (rendered as `\lor`)
  • \neg: Negation (rendered as `\neg`)
  • \exists: Existential quantifier (rendered as `\exists`)
  • \forall: Universal quantifier (rendered as `\forall`)
  • \top: Top (rendered as `\top`)
  • \bot: Bottom (rendered as `\bot`)
  • Parentheses ( ) for grouping terms (rendered as `( )`).
